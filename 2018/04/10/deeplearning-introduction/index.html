Introduction to Deep Learning

Posted on 2018-04-10 | In Computer , Deep Learning

We can start with a simple example: predicting house price.

House Price

Supposing we have a dataset of house price:

x (size of the house) y(price)
30 62
35 83
58 90
79 124
92 150
125 188

The blue line in the plot is to fit the points.

We can write the function in two parts: A linear function and a activation function.

According to the data, we can calculate the slope and the

Linear function: $f(x) = Wx + b = 2.3x - 46$

Activation function: $f(z) = max(z, 0)$, any input smaller than zero will output zero

Combine these two:
$$
f(x) = max(Wx+b, 0)
$$
A neuron is like this function. It accepts the input x, and give the output y.

$f(z) = max(z, 0)$ is called Rectified Linear Unit (ReLU), which is used as activation function.

A Neural Network is to connect neurons with diffrent input dimensons and parameters.

Standard NN, CNN and RNN

Convolutional Neural Network can be used on image problems.

Recurrent Neural Network can be used on audio problems.
